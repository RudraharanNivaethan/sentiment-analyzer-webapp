About This Application

The Sentiment Analysis Web Application is built using Flask, NLTK (VADER), and TextBlob. It enables users to input any sentence or paragraph and instantly receive insights into the sentiment behind the text.

The app combines two key techniques:
  • VADER Sentiment Analyzer: Specially tuned for social media and general text sentiment scoring.
  • TextBlob: Provides subjectivity measurement and additional language processing features.

Visual representation is provided using Matplotlib, so you can see how much of your text is positive, neutral, or negative.
